Welcome to THE GAME!

We hope you liked it!

This game is built without Javascript and only using CSS pseudo-classes. The advantage of this is that even users, who have set their browsers to not use Javascript (for security purposes) can still play.

Future developments will include:

  • A Javascript Feature to randomize the game, so it is different every time.
  • A database to save users progress.
  • Easy, medium, and hard game options.
  • Better gameplay for touchscreen devices.
